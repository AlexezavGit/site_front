Контент-ориентированный дизайн:

Для раздела Data:

Интерактивная карта Украины с тепловой картой охвата услуг (например, области с высоким уровнем PTSD выделены красным).

Графики динамики улучшений (столбчатые диаграммы с % снижения депрессии/тревоги по шкалам PHQ-9/GAD-7).

Для раздела CSR:

Фотографии/иллюстрации партнерских проектов (например, тренинг для психологов от компании X) с подписями: «Партнерство с [Название] → 100 обученных специалистов в 2024».

Для историй успеха:

Анимированные комиксы с анонимизированными персонажами (без лиц), показывающие этапы восстановления.

Критерии качества:

Доступность: Alt-тексты для всех изображений с ключевыми словами (пример: «График снижения симптомов PTSD у клиентов proto.fund»).

Единство стиля: Использовать единую цветовую палитру (например, синий → доверие, зеленый → устойчивость) и шрифты (Sans Serif для четкости).

Культурная чувствительность: Избегать стереотипных изображений (например, не показывать только женщин как жертв).

3. Улучшение пользовательского интерфейса (UI)
Навигация:

Добавить «липкое» меню с разделами: Home, Impact, Methodology, For Donors, Stories.

Внедрить «хлебные крошки» для удобного возврата между страницами.

Создать кнопки-призывы (CTA) с четкими формулировками: «Support Our Mission», «Download Impact Report», «Join as a Practitioner».

Адаптивность:

Оптимизировать все графические элементы для мобильных устройств (схемы → вертикальные, текст → крупнее).

Упростить формы обратной связи: максимум 3 поля для быстрых заявок.

Интерактивность:

Добавить чат-бота с базовыми сценариями («Как помочь?», «Нужна психологическая поддержка?»).

В разделе For Donors — калькулятор вклада («$100 → 10 часов терапии для беженцев»).

Доступность:

Режим высокой контрастности для слабовидящих.

Возможность увеличивать инфографику без потери качества.

Субтитры для всех видео и аудио-материалов.

Примеры интеграции визуалов и текста:
Для страницы Methodology:
Графическая схема:

Заголовок: «Evidence-Based Interventions for War Trauma».

Визуал: 3 колонки с иконками:

Assessment (анкета + шкала PTSD).

Therapy (КПТ, EMDR, группы поддержки).

Monitoring (график с динамикой симптомов).

Подпись: «Все методы одобрены ВОЗ и адаптированы для конфликтных зон».

Для страницы Impact:
Интерактивная дашборд:

Фильтры: регион, возраст, тип терапии.

Визуал: карта + график + цитата из отчета («За 2023 год 80% клиентов вернулись к работе»).

Кнопка «Download Full Dataset» (в формате CSV/PDF для доноров).

Критерии успеха:
Пользовательский опыт:

Уменьшение времени нахождения в меню до 3 кликов для целевых действий (донорство, запрос помощи).

Увеличение времени просмотра страниц на 40% за счет интерактивных элементов.

Донорский интерес:

Рост скачивания документов (логическая рамка, бюджет) на 50%.

Упоминание визуалов сайта в презентациях партнеров (например, UNHCR).

Доступность:

Соответствие стандарту WCAG 2.1 AA (контраст, навигация с клавиатуры).

Рекомендуемые инструменты и ресурсы:
Для создания инфографики: Canva, Adobe Illustrator, Piktochart.

Для интерактивных дашбордов: Tableau Public, Google Data Studio.
